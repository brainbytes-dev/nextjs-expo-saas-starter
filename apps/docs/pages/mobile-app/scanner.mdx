# Barcode-Scanner

Der Scanner ist das Herzstück der mobilen App. Er erkennt alle gängigen Barcode- und QR-Code-Formate in Echtzeit.

## Scanner öffnen

Tippen Sie in der Navigation unten auf das **Scanner-Symbol** (Kamera-Icon). Der Scanner öffnet sich im Vollbild.

Alternativ öffnet sich der Scanner automatisch an folgenden Stellen:
- Beim Hinzufügen von Lieferschein-Positionen
- Beim Erfassen von Inventur-Ist-Beständen
- Beim Aus- oder Einchecken von Werkzeugen

## Unterstützte Formate

| Format | Typische Verwendung |
|--------|-------------------|
| EAN-13 / EAN-8 | Konsumgüter, Handelswaren |
| Code 128 | Industrie, Logistik |
| Code 39 | Ältere Etiketten |
| QR Code | **ZEN**TORY-interne Artikel und Standorte |
| Data Matrix | Elektronikbauteile |
| PDF417 | Fahrzeugpapiere, Behörden |

## Scan-Tipps

- **Licht**: Stellen Sie sicher, dass der Barcode gut beleuchtet ist. Bei schlechtem Licht aktivieren Sie das **Taschenlampen-Symbol** im Scanner.
- **Abstand**: Halten Sie das Gerät 10–30 cm vom Barcode entfernt.
- **Winkel**: Halten Sie das Gerät möglichst parallel zum Barcode.
- **Beschädigte Codes**: Falls ein Code stark beschädigt ist, können Sie die **Nummer manuell eingeben** (Tastatur-Symbol im Scanner).

## Ergebnis-Aktionen

Nach dem Scan eines bekannten Artikels stehen folgende Aktionen zur Auswahl:

- **Details anzeigen**: Öffnet die Artikeldetailseite
- **Auschecken / Einchecken**: Nur bei Werkzeugen
- **Zu Lieferschein hinzufügen**: Zur aktuell offenen Bestellung
- **Bestand anpassen**: Schnelle Bestandskorrektur

## Scan-Verlauf

Der Scanner speichert die letzten 20 gescannten Artikel im lokalen Verlauf. Tippen Sie auf **Verlauf** (Uhr-Symbol) um einen Artikel ohne erneutes Scannen aufzurufen.

## Mehrfach-Scan

Im Mehrfach-Scan-Modus (aktivierbar über das **Stapel-Symbol**) werden mehrere Artikel hintereinander gescannt und als Gruppe verarbeitet. Nützlich für:

- Kommissionieren einer Lieferung
- Aufnehmen des Fahrzeugbestands
- Massenausbuchung für eine Baustelle

## Manuelle Eingabe

Falls der Barcode nicht lesbar ist:

1. Tippen Sie auf das **Tastatur-Symbol** im Scanner.
2. Geben Sie EAN, Artikelnummer oder Artikelname ein.
3. Tippen Sie auf **Suchen**.

## Offline-Scanning

Der Scanner funktioniert auch ohne Internetverbindung. Gescannte Aktionen werden gespeichert und bei nächster Verbindung synchronisiert. Mehr dazu unter [Offline-Modus](/mobile-app/offline).
