# Push-Benachrichtigungen

Die mobile App kann Sie über wichtige Ereignisse informieren, auch wenn die App nicht geöffnet ist.

## Benachrichtigungstypen

| Typ | Auslöser | Empfänger |
|-----|---------|-----------|
| Mindestbestand unterschritten | Bestand fällt unter Schwellenwert | Administratoren, Manager |
| Wartung fällig | 30 / 7 / 1 Tag(e) vor Termin | Zuständige Person, Administratoren |
| Werkzeug überfällig | Rückgabedatum überschritten | Ausleihende Person, Manager |
| Einladung angenommen | Neues Mitglied hat Einladung akzeptiert | Administrator |
| Inventur gestartet | Neue Inventur erstellt | Alle Mitglieder |
| Inventur abgeschlossen | Inventur fertiggestellt | Administratoren |
| Import abgeschlossen | Grossimport fertig | Initiierender Benutzer |

## Benachrichtigungen konfigurieren

1. Gehen Sie in der App zu **Profil → Benachrichtigungen**.
2. Aktivieren oder deaktivieren Sie einzelne Typen.
3. Wählen Sie für Zeitbasierte Erinnerungen den Vorlaufzeitraum (7, 14 oder 30 Tage).

Globale Einstellungen für die gesamte Organisation finden Sie in der Web-App unter [Einstellungen → Benachrichtigungen](/einstellungen/benachrichtigungen).

## Benachrichtigungen deaktivieren

Um alle Push-Benachrichtigungen zu deaktivieren:

**iOS:** Einstellungen → **ZEN**TORY → Benachrichtigungen → deaktivieren

**Android:** Einstellungen → Apps → **ZEN**TORY → Benachrichtigungen → deaktivieren

> **Hinweis:** Deaktivierte Benachrichtigungen auf Geräteebene überschreiben alle App-Einstellungen. Die Ereignisse werden weiterhin per E-Mail gesendet (falls E-Mail-Benachrichtigungen aktiviert sind).
