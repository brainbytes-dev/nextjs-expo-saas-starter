# Integrationen

**ZEN**TORY lässt sich mit verschiedenen externen Systemen verbinden.

## Verfügbare Integrationen

### Zapier
Verbinden Sie **ZEN**TORY mit über 5000 Apps ohne Programmierung:
1. Gehen Sie zu **Einstellungen → Integrationen → Zapier**.
2. Klicken Sie auf **Verbinden** und authentifizieren Sie sich bei Zapier.
3. Erstellen Sie Zaps z.B.: «Wenn Bestand unter Minimum → Slack-Nachricht senden» oder «Wenn neuer Lieferschein → Google Sheets Zeile hinzufügen».

### Make (ehemals Integromat)
Ähnlich wie Zapier, mit visueller Workflow-Erstellung. Unter **Einstellungen → Integrationen → Make**.

### Microsoft Teams / Slack
Senden Sie Benachrichtigungen direkt in einen Teams-Kanal oder Slack-Channel:
1. Wählen Sie die gewünschte App.
2. Klicken Sie auf **Verbinden** und autorisieren Sie den Zugriff.
3. Wählen Sie den Kanal.
4. Wählen Sie, welche Ereignisse gemeldet werden sollen.

### CSV-Export (automatisch)
Planen Sie automatische CSV-Exporte an eine E-Mail-Adresse oder FTP-Server:
1. Gehen Sie zu **Einstellungen → Integrationen → Geplante Exporte**.
2. Wählen Sie: Bestandsliste, Lieferscheine oder Werkzeug-Buchungen.
3. Legen Sie Häufigkeit und Empfänger fest.
