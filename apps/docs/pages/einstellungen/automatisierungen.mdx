# Automatisierungen

Hier finden Sie eine Übersicht aller Automatisierungsfunktionen in **ZEN**TORY. Die detaillierte Konfiguration jeder Funktion finden Sie in der verlinkten Dokumentation.

## Übersicht der Automatisierungsfunktionen

| Funktion | Beschreibung | Plan |
|---|---|---|
| [Workflow-Regeln](/automatisierung/workflow-regeln) | Wenn-Dann-Regeln für beliebige Ereignisse | Business+ |
| [Genehmigungen](/automatisierung/genehmigungen) | Kritische Aktionen genehmigungspflichtig machen | Business+ |
| [Auto-Nachbestellung](/automatisierung/auto-nachbestellung) | Bestände automatisch auffüllen | Business+ |
| [Geplante Berichte](/automatisierung/geplante-berichte) | Regelmässige E-Mail-Berichte | Business+ |
| [Webhooks](/einstellungen/webhooks) | Externe Systeme bei Ereignissen benachrichtigen | Starter+ |

## Schnellzugriff

Alle Automatisierungseinstellungen finden Sie zentral unter **Automatisierung** in der Hauptnavigation:

- **Workflow-Regeln** — Regeln anzeigen, erstellen, testen
- **Genehmigungen** — Ausstehende Genehmigungen bearbeiten
- **Nachbestellvorschläge** — Bestellvorschläge prüfen und ausführen
- **Geplante Berichte** — Berichte konfigurieren und verwalten
- **Protokoll** — Alle ausgeführten Automatisierungsaktionen

## Automatisierungen deaktivieren

Falls Sie alle Automatisierungen vorübergehend deaktivieren müssen (z.B. während einer Datenmigration):

1. Gehen Sie zu **Einstellungen → Automatisierungen → Notfallstopp**.
2. Aktivieren Sie **Alle Automatisierungen pausieren**.
3. Laufende Aktionen werden abgeschlossen; neue Auslöser werden ignoriert.
4. Deaktivieren Sie den Notfallstopp wenn bereit.

> **Warnung:** Im Pausemodus werden keine Alerts, keine Nachbestellungen und keine Geplanten Berichte ausgeführt. Verwenden Sie diese Option nur wenn nötig.
