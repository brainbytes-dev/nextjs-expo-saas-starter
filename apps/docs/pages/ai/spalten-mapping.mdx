# Import AI-Mapping

Beim CSV- oder Excel-Import erkennt **ZEN**TORY Ihre Spalten automatisch und ordnet sie den richtigen Feldern zu — auch wenn die Spaltenbezeichnungen von der erwarteten Vorlage abweichen.

## Was das AI-Mapping löst

Jedes Unternehmen benennt seine Tabellenspalten anders. Statt «Artikelnummer» vielleicht «Art-Nr.», «SKU» oder «Materialnummer». Das AI-Mapping versteht semantische Ähnlichkeiten und schlägt die passende Zuordnung vor.

> **Hinweis:** Das AI-Mapping ist ab dem **Starter-Plan** verfügbar. Bei einfachen Spaltenbezeichnungen greift das regelbasierte Fallback und benötigt kein AI-Kontingent.

## Import mit AI-Mapping — Schritt für Schritt

1. Gehen Sie zu **Materialien → Import** (oder **Werkzeuge → Import**).
2. Laden Sie Ihre CSV- oder Excel-Datei hoch.
3. **ZEN**TORY zeigt eine **Vorschau** der ersten 5 Zeilen Ihrer Datei.
4. Unter der Vorschau erscheint das **Spalten-Mapping**:
   - Ihre Spaltenbezeichnung (links)
   - Vorgeschlagenes **ZEN**TORY-Feld (rechts, mit Konfidenz-Badge)
5. Prüfen Sie jeden Vorschlag:
   - **Grüner Badge**: Hohe Konfidenz, Zuordnung wahrscheinlich korrekt
   - **Gelber Badge**: Mittlere Konfidenz, bitte prüfen
   - **Grauer Badge**: Keine Erkennung — manuell aus Dropdown wählen
6. Korrigieren Sie bei Bedarf einzelne Zuordnungen per Dropdown.
7. Klicken Sie auf **Mapping übernehmen und importieren**.

## Unterstützte Felder

| **ZEN**TORY-Feld | Erkannte Spaltenbezeichnungen (Beispiele) |
|---|---|
| Name | Bezeichnung, Artikel, Produktname, Description |
| Artikelnummer | Art-Nr., SKU, Materialnummer, Item No. |
| EAN | Barcode, GTIN, EAN-13, UPC |
| Kategorie | Kategorie, Gruppe, Type, Warengruppe |
| Einheit | Einheit, Unit, UOM, Mengeneinheit |
| Bestand | Bestand, Menge, Qty, Quantity, Stock |
| Mindestbestand | Min-Bestand, Reorder Point, Meldebestand |
| Standort | Lager, Location, Standort, Lagerort |
| Preis | Preis, Einkaufspreis, Cost, Price |

## Regelbasiertes Fallback

Falls KI nicht verfügbar ist, verwendet **ZEN**TORY eine Regelbibliothek mit über 200 vordefinierten Spalten-Synonymen. Das Fallback funktioniert vollständig offline und ohne externe API-Aufrufe.

> **Tipp:** Laden Sie die **Importvorlage** (unter **Materialien → Import → Vorlage herunterladen**) herunter — dann entfällt das Mapping vollständig.

## Mapping als Regel speichern

Wenn Sie regelmässig Exporte aus demselben System importieren, können Sie das fertige Mapping als **Regel speichern**:

1. Klicken Sie nach dem Mapping auf **Regel speichern**.
2. Vergeben Sie einen Namen (z.B. «SAP-Export Lager A»).
3. Beim nächsten Import wählen Sie die Regel aus — das Mapping wird automatisch angewendet.

Gespeicherte Mapping-Regeln verwalten Sie unter **Einstellungen → Import-Regeln**.
