# Registrierung & Login

In diesem Abschnitt erfahren Sie, wie Sie ein Konto für **ZEN**TORY erstellen und sich anmelden.

## Konto erstellen

1. Navigieren Sie zu [app.zentory.ch/signup](https://app.zentory.ch/signup).
2. Geben Sie Ihren vollständigen Namen, Ihre geschäftliche E-Mail-Adresse und ein sicheres Passwort ein.
3. Akzeptieren Sie die Nutzungsbedingungen und Datenschutzerklärung.
4. Klicken Sie auf **Konto erstellen**.
5. Prüfen Sie Ihr E-Mail-Postfach: Sie erhalten eine Bestätigungs-E-Mail. Klicken Sie auf den enthaltenen Link, um Ihre Adresse zu verifizieren.

> **Tipp:** Verwenden Sie Ihre geschäftliche E-Mail-Adresse. Kollegen, die sich mit der gleichen Domain registrieren, können später einfacher zu Ihrem Unternehmen hinzugefügt werden.

## Anmelden

1. Öffnen Sie [app.zentory.ch/login](https://app.zentory.ch/login).
2. Geben Sie Ihre E-Mail-Adresse und Ihr Passwort ein.
3. Optional: Aktivieren Sie **Angemeldet bleiben**, damit die Session 30 Tage aktiv bleibt.
4. Klicken Sie auf **Anmelden**.

### Passwort vergessen

Falls Sie Ihr Passwort vergessen haben:

1. Klicken Sie auf der Login-Seite auf **Passwort vergessen**.
2. Geben Sie Ihre E-Mail-Adresse ein.
3. Sie erhalten einen Link zum Zurücksetzen des Passworts. Dieser Link ist 24 Stunden gültig.
4. Klicken Sie auf den Link und vergeben Sie ein neues Passwort (mindestens 8 Zeichen, davon 1 Grossbuchstabe und 1 Zahl).

## Single Sign-On (SSO)

Unternehmen mit einem Business- oder Enterprise-Abo können sich via SSO mit ihrem bestehenden Identitätsprovider (Microsoft Entra ID, Google Workspace, Okta) anmelden. Die Konfiguration erfolgt unter [Einstellungen → SSO](/einstellungen/sso).

## Zwei-Faktor-Authentifizierung (2FA)

Für erhöhte Sicherheit empfehlen wir die Aktivierung von 2FA:

1. Gehen Sie zu **Einstellungen → Profil & Konto**.
2. Klicken Sie unter **Sicherheit** auf **2FA aktivieren**.
3. Scannen Sie den QR-Code mit einer Authenticator-App (z.B. Google Authenticator, Authy).
4. Geben Sie den 6-stelligen Code zur Bestätigung ein.

> **Hinweis:** Administratoren können 2FA für alle Mitglieder einer Organisation erzwingen. Mehr dazu unter [Team & Rollen](/einstellungen/team).

## Abmelden

Klicken Sie oben rechts auf Ihr Profilbild und wählen Sie **Abmelden**. Alle lokalen Session-Daten werden gelöscht.
