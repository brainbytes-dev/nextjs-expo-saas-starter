# Team einladen

Laden Sie Mitarbeitende ein, damit diese ebenfalls auf **ZEN**TORY zugreifen können.

## Mitglied einladen

1. Gehen Sie zu **Einstellungen → Team & Rollen**.
2. Klicken Sie auf **Mitglied einladen**.
3. Geben Sie die E-Mail-Adresse der Person ein.
4. Wählen Sie eine **Rolle** (siehe unten).
5. Optional: Fügen Sie eine persönliche Nachricht zur Einladungs-E-Mail hinzu.
6. Klicken Sie auf **Einladung senden**.

Die eingeladene Person erhält eine E-Mail mit einem Link. Der Link ist **7 Tage gültig**. Die Person muss beim ersten Login ein Passwort setzen (oder sich via SSO anmelden, falls konfiguriert).

> **Tipp:** Sie können mehrere Personen gleichzeitig einladen, indem Sie E-Mail-Adressen kommagetrennt eingeben.

## Rollen und Berechtigungen

| Rolle | Beschreibung | Typische Personen |
|-------|--------------|-------------------|
| **Administrator** | Vollzugriff inkl. Einstellungen, Abonnement, Benutzerverwaltung | Geschäftsführer, IT-Verantwortliche |
| **Manager** | Kann alle Inhalte erstellen und bearbeiten, Berichte exportieren, aber keine Systemeinstellungen ändern | Lagerleiter, Teamleiter |
| **Mitglied** | Kann Materialien anzeigen, Lieferscheine erstellen, Inventur durchführen | Mitarbeitende, Monteure |
| **Lesezugriff** | Kann nur lesen, nichts erstellen oder bearbeiten | Externe Prüfer, Revisoren |

> **Hinweis:** Berechtigungen können unter [Einstellungen → Team & Rollen](/einstellungen/team) individuell angepasst werden (nur Enterprise-Abo).

## Ausstehende Einladungen verwalten

Unter **Einstellungen → Team & Rollen → Ausstehende Einladungen** sehen Sie alle noch nicht angenommenen Einladungen. Sie können:

- Eine Einladung **erneut senden** (neuer Link mit 7 Tage Laufzeit)
- Eine Einladung **widerrufen** (der Link wird sofort ungültig)

## Mitglied entfernen

1. Gehen Sie zu **Einstellungen → Team & Rollen**.
2. Klicken Sie neben dem Mitglied auf **Optionen (⋯) → Entfernen**.
3. Bestätigen Sie die Aktion.

Das Mitglied wird sofort abgemeldet und hat keinen Zugriff mehr. Alle von der Person erstellten Inhalte bleiben erhalten und werden dem entfernenden Administrator zugewiesen.

## Lizenzgrenzen

Die Anzahl der Benutzer hängt von Ihrem Abo-Plan ab:

| Plan | Benutzer |
|------|----------|
| Starter | bis 3 |
| Business | bis 15 |
| Enterprise | unbegrenzt |

Falls Sie die Grenze erreichen, werden Sie beim Einladen darauf hingewiesen. Sie können Ihr Abo jederzeit unter **Einstellungen → Abonnement** upgraden.
