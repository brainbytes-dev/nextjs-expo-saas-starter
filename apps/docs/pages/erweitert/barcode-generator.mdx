# Barcode-Generator

Für Artikel ohne EAN oder Seriennummer können Sie in **ZEN**TORY eigene Barcodes generieren und drucken.

## Wozu eigene Barcodes?

Nicht alle Artikel haben einen aufgedruckten Barcode — zum Beispiel selbst hergestellte Produkte, Werkzeuge ohne Etikett oder Verbrauchsmaterialien in Grossgebinden. Mit dem integrierten Generator erstellen Sie standardisierte Barcodes im **Code-128-Format**, die von allen gängigen Scannern gelesen werden.

## Format der generierten Barcodes

**ZEN**TORY generiert Barcodes nach folgendem Schema:

```
LA-{ORG}-{000001}
```

- `LA` — Präfix für **ZEN**TORY (unterscheidet von externen EANs)
- `{ORG}` — Kürzel Ihrer Organisation (2–4 Zeichen, aus Einstellungen)
- `{000001}` — Fortlaufende 6-stellige Nummer, auto-inkrementiert

Beispiel: `LA-MBK-000042`

## Einzelnen Barcode generieren

1. Öffnen Sie einen Artikel (Material oder Werkzeug).
2. Klicken Sie auf **Bearbeiten** und scrollen Sie zu **Barcode / EAN**.
3. Falls kein Barcode hinterlegt ist, erscheint die Schaltfläche **Barcode generieren**.
4. **ZEN**TORY weist automatisch den nächsten freien Code zu.
5. Speichern Sie den Artikel.

## Batch-Generierung

Für mehrere Artikel gleichzeitig:

1. Öffnen Sie **Materialien** oder **Werkzeuge**.
2. Filtern Sie nach **Barcode: leer** (Filterleiste oben).
3. Aktivieren Sie die Checkboxen und wählen Sie die gewünschten Artikel.
4. Klicken Sie in der Aktionsleiste auf **Barcode generieren**.
5. **ZEN**TORY weist jedem Artikel automatisch einen eindeutigen Code zu.

> **Hinweis:** Die Batch-Generierung ist auf **200 Artikel** pro Vorgang begrenzt.

## Vorschau und Druck

Nach der Generierung können Sie die Barcodes sofort drucken:

1. Wählen Sie einen oder mehrere Artikel mit generierten Barcodes aus.
2. Klicken Sie auf **Etiketten drucken**.
3. Wählen Sie das **Etikettenformat** (Avery, Zebra, Brother, Dymo oder individuell).
4. Die Vorschau zeigt das Layout mit Barcode, Artikelname und Artikelnummer.
5. Klicken Sie auf **Drucken**.

Jedes Etikett enthält:
- Den Barcode als Strichcode-Grafik
- Die lesbare Barcode-Nummer darunter
- Den Artikelnamen (optional)
- Die Artikelnummer (optional)

## Barcode-Kollision verhindern

**ZEN**TORY prüft bei jeder Generierung, ob der Code bereits vergeben ist. Da das Format organisations-spezifisch ist (`LA-{ORG}-`), entstehen keine Konflikte zwischen verschiedenen Organisationen im Multi-Unternehmen-Modus.
