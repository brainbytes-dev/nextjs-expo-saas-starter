# Command Palette

Die Command Palette ist die schnellste Art, in **ZEN**TORY zu navigieren. Mit einem einzigen Tastaturkürzel erreichen Sie jeden Bereich, jeden Artikel und jede Aktion — ohne die Hände von der Tastatur zu nehmen.

## Öffnen und Schliessen

| Aktion | Shortcut |
|--------|----------|
| Command Palette öffnen | `Cmd+K` (Mac) / `Ctrl+K` (Windows/Linux) |
| Schliessen | `Esc` |
| Nächster Eintrag | `↓` |
| Vorheriger Eintrag | `↑` |
| Auswahl bestätigen | `Enter` |

## Suche

Beginnen Sie einfach zu tippen — die Palette durchsucht gleichzeitig:

- **Materialien**: Name, Artikelnummer, EAN
- **Werkzeuge**: Name, Seriennummer, Status
- **Standorte**: Name, Typ
- **Lieferscheine**: Nummer, Lieferant, Empfänger
- **Einstellungen**: Alle Konfigurationsseiten

Treffer werden in Gruppen unterteilt und nach Relevanz sortiert. Mit `Tab` können Sie zwischen den Gruppen wechseln.

## Navigations-Shortcuts

Geben Sie einen Buchstabencode ein, um direkt zu einem Bereich zu springen:

| Shortcut | Ziel |
|----------|------|
| `g h` | Dashboard (Home) |
| `g m` | Materialien-Übersicht |
| `g w` | Werkzeuge-Übersicht |
| `g s` | Standorte |
| `g l` | Lieferscheine |
| `g i` | Inventur |
| `g r` | Berichte |
| `g e` | Einstellungen |

## Aktions-Shortcuts

| Shortcut | Aktion |
|----------|--------|
| `c m` | Neues Material erstellen |
| `c t` | Neues Werkzeug erstellen |
| `c l` | Neuen Lieferschein erstellen |
| `c s` | Neuen Standort erstellen |
| `?` | Alle Shortcuts anzeigen |

## Aktionen ausführen

Neben der Navigation können Sie auch direkte Aktionen aufrufen:

- **«Neues Material»** — öffnet das Erstellungsformular
- **«Inventur starten»** — startet den Inventur-Workflow
- **«Bericht exportieren»** — öffnet den Export-Dialog
- **«Werkzeug einchecken»** — öffnet den Check-in-Dialog
- **«Einstellungen: Team»** — springt direkt zur Team-Seite

Tippen Sie einfach einen Teil des Aktionsnamens — **ZEN**TORY zeigt alle passenden Befehle.

## Zuletzt verwendet

Die Command Palette merkt sich die letzten 10 aufgerufenen Einträge und zeigt diese beim Öffnen ohne Eingabe an. So erreichen Sie häufig genutzte Seiten und Objekte mit nur zwei Tastendrücken.

## Tastaturkürzel-Übersicht

Drücken Sie `?` (Fragezeichen) auf jeder Seite, um eine Übersicht aller aktiven Shortcuts anzuzeigen. Diese Liste ist kontextsensitiv — auf der Materialliste werden andere Shortcuts angezeigt als im Lieferschein-Editor.

> **Tipp:** Die Command Palette funktioniert überall in der Web-App, ausser wenn ein Eingabefeld aktiv fokussiert ist. Drücken Sie `Esc` um den Fokus zu verlassen und dann `Cmd+K`.
